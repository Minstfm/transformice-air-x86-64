1. Get the latest Adobe AIR SDK and [here](https://airsdk.harman.com/api/versions/51.2.2.6/sdks/AIRSDK_Windows.zip?license=accepted) extract it
2. Get the StandAlone files [here](https://github.com/Minstfm/transformice-air-x86-64/archive/refs/heads/main.zip) and extract it
3. Run for debug standalone `AIRSDK\bin\adl64 -runtime AIRSDK\runtimes\air\win64 -nodebug folder\application.xml`
